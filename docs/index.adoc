== Introduction

Explorer is an enterprise search solution which simplifies collecting and searching data from multiple sources.

image::explorer.png[Explorer,768]

It consists of:

. An <<admin#,administration>> application.
. A <<library#,library>> to implement <<client#,Client>> and <<collector#,Collector>> applications.
